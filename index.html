Google Drive web hosting server:
More file hosting storage space for the website so it uses less of the limited-availability free disk space.


examples:

/f.php/pic/

/f.php/Galaxies.html



Description feature:
/f.php/[shared folder or file id]/dscr.txt
/f.php/[shared folder id]/path/to/file.jpeg/dscr.txt
Append "/dscr.txt" to the end of any file path to see that Google-Drive file's description attribute if it has one.
User "/dscr.json" or "/dscr.html" to get JSON or HTML formatted descriptions.
